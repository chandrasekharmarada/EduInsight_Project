🎓 Student Analytics using Machine Learning

A machine learning system that analyzes student academic 
data to predict performance outcomes and identify at-risk 
students early.

Key Features:
- Predicts student academic performance using ML models
- Identifies at-risk students before results are declared  
- Visualizes key factors affecting learning outcomes
- Transforms raw educational data into actionable insights

Tech Stack: Python | Scikit-learn | Pandas | NumPy | Matplotlib

Impact: Enables educators to make data-driven decisions
and provide timely intervention for struggling students.
